## 1. 概述

### 1.1 什么是 Topo

Topo 是一款 Windows 桌面端网络拓扑绘制工具，专为网络工程师和 IT 管理人员设计。你可以通过拖拽设备、连接线路、配置属性，快速绘制专业的网络拓扑图。

**核心能力：**

- 内置 16 类网络设备和 42+ 预置型号（覆盖 H3C、华为、Cisco、锐捷、Aruba 等主流厂商）
- 支持以太网、光纤、堆叠、隧道、无线五种连线类型，每种支持独立的动画效果和样式定制
- 完整的机柜系统：支持标准 U 位管理、设备入柜、正面/背面视图切换
- 三种导出格式：PNG（静态截图）、PDF（可打印文档）、GIF（带动画效果）
- 中英文双语界面、默认/鎏金双主题
- 撤销/重做、复制/粘贴、分组、模板等高效编辑功能

### 1.2 适用平台

- Windows 10 / Windows 11（64 位）
- 提供安装包（.exe）和便携版两种发布形式

### 1.3 启动应用

安装完成后，双击桌面快捷方式或从开始菜单启动 Topo。启动后默认显示空白画布，可以直接开始工作。

### 2.2 左侧边栏——设备库

左侧边栏分为三个区域：

**搜索栏**（顶部）
- 输入关键词实时过滤设备列表
- 搜索范围：设备型号、厂商、分类、描述

**机柜区**
- 提供 4 种预置机柜尺寸：壁挂式（≤12U）、小型（13-22U）、中型（23-36U）、标准（42U）
- 直接拖拽到画布即可放置机柜

**设备分类区**（中间，可折叠展开）
- 16 个设备分类，每类显示设备数量
- 点击分类标题展开/折叠设备列表
- 每个设备项显示：颜色标识 + 厂商 + 型号 + 端口摘要
- **悬停**：弹出设备详细信息提示框
- **右键**：编辑设备信息 / 删除设备
- **拖拽**：拖动到画布上创建节点

**底部**
- 「添加自定义设备」按钮：打开设备编辑对话框
- 设备总数和分类数统计

### 2.3 中央画布

中央画布是主要的拓扑绘制区域，基于 React Flow 引擎实现：

- **无限画布**：可任意方向拖拽平移，无边界限制
- **缩放**：鼠标滚轮缩放（10%-300%），右下角有内置缩放控件
- **平移**：鼠标中键拖拽，或按住右键拖拽
- **框选**：在空白区域按住鼠标左键拖拽，框选多个节点
- **MiniMap**：左下角小地图，概览整个拓扑布局

### 2.4 右侧属性面板

属性面板根据当前选中的元素动态切换显示内容：

| 选中状态 | 面板内容 |
|----------|----------|
| 未选中 | 空白提示 |
| 选中 2+ 节点 | 节点数量 + 操作提示 |
| 选中设备节点 | 节点颜色、名称、厂商、型号、端口、照片、IP、备注等 |
| 选中机柜节点 | 机柜名称、U 位统计、视图切换、尺寸信息 |
| 选中机柜内设备 | U 高调整、电源数量、颜色、名称 |
| 选中连线 | 连线类型、路径样式、动画效果、颜色、粗细、端口、IP 等 |

### 2.5 状态栏

窗口底部的状态栏提供实时信息：

| 位置 | 内容 |
|------|------|
| 左侧 | 缩放百分比 · 设备数 · 连线数 · 选中数量 |
| 右侧 | 快捷键提示：`Ctrl+G 分组` `Ctrl+F 搜索` `Ctrl+C/V 复制粘贴` `Esc 取消选中` |

## 3. 快速入门

按照以下步骤，5 分钟内完成你的第一个拓扑图：

### 步骤 1：添加设备

1. 在左侧边栏找到「交换机」分类，点击展开
2. 找到 H3C S5130-28P，按住鼠标左键**拖拽到画布中央**
3. 松开鼠标，设备节点出现在画布上
4. 再拖入一个「终端-PC」设备

### 步骤 2：连接设备

1. 将鼠标移到交换机节点上，设备边缘会出现小圆点（连接端口）
2. 按住小圆点，拖拽到 PC 节点上
3. 松开鼠标，一条连线就创建好了
4. 默认创建的是以太网连线（实线），系统会自动分配端口

### 步骤 3：调整属性

1. **单击**交换机节点，右侧属性面板显示设备属性
2. 可以在「设备名称」输入框中修改节点显示名称，按 Enter 确认
3. **单击**连线，右侧属性面板切换为连线属性
4. 试试将「连线类型」改为「光纤」——连线变为虚线
5. 试试将「动画效果」改为「粒子流动」——连线上出现流动粒子

### 步骤 4：保存文件

1. 点击工具栏「保存」按钮（或按 Ctrl+S）
2. 在弹出对话框中选择保存位置和文件名
3. 文件以 `.topo.json` 格式保存，下次可以直接打开

> 💡 **提示**：右键点击空白画布，选择「适应画布」可自动调整视图。

### 4.2 添加设备到画布

**方式一：拖拽添加**
1. 在左侧边栏展开目标分类
2. 找到目标设备型号
3. 按住拖拽到画布上任意位置释放

**方式二：拖入机柜**
- 将设备直接拖到机柜节点上方，设备会自动放入机柜的第一个空闲 U 位

### 4.3 自定义添加设备

如果内置设备库没有你需要的型号，可以自行添加：

1. 点击左侧边栏底部的「添加自定义设备」按钮
2. 在弹出的对话框中填写：
   - **分类**（必选）：从下拉列表选择设备类型
   - **厂商**（必选）：从下拉列表选择或点击「+ 新厂商」创建
   - **型号**（必填）：输入设备型号名称
   - **描述**（可选）：设备补充说明
   - **端口配置**：设置 RJ45 端口数 / SFP 光纤端口数 / 10G 光纤端口数（0-256），右侧有实时端口串预览
   - **设备照片**（可选）：上传一张设备实拍图，仅在属性面板中预览，不影响画布 SVG 渲染
3. 点击「创建」保存

### 4.4 编辑/删除设备

- **编辑**：在侧边栏右键点击设备 → 选择「编辑设备信息」→ 打开编辑对话框 → 修改后点击「更新」
- **删除**：在侧边栏右键点击设备 → 选择「删除设备」→ 确认删除

> ⚠️ 删除设备会从数据库中永久移除该型号，但不会影响画布上已有的节点。

### 4.5 侧边栏设备搜索

1. 点击侧边栏顶部的搜索框
2. 输入关键词（型号/厂商/分类/描述）
3. 搜索结果立即显示（300ms 防抖），显示匹配数量
4. 清空搜索框恢复分类视图
5. 搜索结果中的设备同样支持拖拽、右键操作

| 适应画布 | 工具栏 ⊞ 按钮 / Ctrl+0 |

### 5.7 画布平移

| 方法 | 说明 |
|------|------|
| 鼠标中键拖拽 | 按住中键拖动平移画布 |
| 鼠标右键拖拽 | 按住右键拖动平移画布 |

画布左下角自动显示一个小地图（MiniMap），提供拓扑的鸟瞰视图：

- 深色区域表示当前视口范围
- 直接在小地图上拖拽可快速导航
- 颜色跟随当前主题变化

## 6. 连线系统

### 6.1 创建连线

1. 鼠标悬停在源设备上，设备边缘出现连接端口（小圆点 Handle）
2. 按住一个端口圆点，拖拽到目标设备上
3. 松开鼠标，连线创建完成
4. 系统自动分配端口名称并设置连接类型

**特殊端口自动识别：**
- 从 Stack 端口拖出 → 自动创建 Stack（堆叠）连线
- 从 Tunnel 端口拖出 → 自动创建 Tunnel（隧道）连线
- 从 WLAN 端口拖出 → 自动创建 Wireless（无线）连线

### 6.2 连线类型

| 类型 | 视觉样式 | 默认颜色 | 用途 |
|------|----------|----------|------|
| Ethernet（以太网） | 实线 | 蓝色系 | 铜缆/RJ45 连接 |
| Fiber（光纤） | 虚线 | 青色系 | SFP/光纤连接 |
| Stack（堆叠） | 粗金线（7px） | 金色 | 交换机堆叠互联 |
| Tunnel（隧道） | 阶梯路径（6px） | 青色 | SDWAN CPE 隧道至 SDWAN 节点 |
| Wireless（无线） | 信号波纹 | 绿色系 | WLAN 或 AP 无线连接 |

**连线验证规则：**
- Stack 端口只能连接 Stack 端口
- WLAN 只能连接 WLAN 或无线 AP
- Tunnel 只能连接 CPE Tunnel 端口与 SDWAN Node 的 CP-LEFT/CP-RIGHT

连线类型可在属性面板中随时更改。

### 6.3 连线路径样式

单击连线，在右侧属性面板的「连线形式」下拉框中选择：

| 样式 | 说明 |
|------|------|
| 自适应（Adaptive） | 默认贝塞尔曲线，自动调整弧度 |
| 直线（Straight） | 最短路径直线连接 |
| 阶梯（Step） | 直角弯折路径 |

**阶梯路径额外调节：**
- 弯折高度（Elbow Height）：10-400px 滑块，控制竖线高度
- 水平偏移（Elbow Offset）：-200 到 +200px，控制弯折点向左或向右偏移
- 「重置弯折高度」和「居中偏移」按钮可恢复默认值

### 6.4 动画效果

在属性面板的「显示状态」下拉框中选择：

| 效果 | 说明 |
|------|------|
| 静态（Static） | 无动画，纯静态线条 |
| 粒子流动（Particle） | 沿连线方向流动的彩色粒子 |
| 光带流动（Glow） | 沿连线方向移动的光带 |
| 信号波纹（Wave） | 辐射状波纹扩散（无线连接默认） |

### 6.5 连线外观自定义

选中连线后，属性面板提供以下调节项：

| 参数 | 范围 | 说明 |
|------|------|------|
| 线缆粗细 | 1-10px（0.5步进） | 滑块调节 |
| 线缆颜色 | 任意颜色 | 颜色选择器 |
| 动画速度 | 0.5x-4x（0.25步进） | 显示 Fast/Normal/Slow 标签 |
| 粒子大小 | 2-12px（0.5步进） | 仅在粒子流动模式显示 |
| 特效颜色 | 任意颜色 | 动画粒子/光带/波纹的颜色 |

### 6.6 连线方向

- 每条连线默认带有方向箭头
- 属性面板中「方向」有正向/反向两个按钮，点击切换
- 方向影响动画流动方向和箭头朝向

### 6.7 端口管理

选中连线后可以精细管理两端端口：

- **源端口**：显示一个输入框（自动分配）和「自动检测」按钮
- **目标端口**：同上
- **下拉选择**：旁边有端口下拉菜单，列出设备所有可用端口
  - 已被其他连线使用的端口标记为禁用（灰色）
  - 选择后自动更新端口标签
- **端口标签拖拽**：画布上的端口标签（源蓝色/目标橙色）可独立拖拽调整位置

### 6.8 接口 IP 配置

- **本地 IP**：源端接口 IP 地址
- **对端 IP**：目标端接口 IP 地址
- IP 标签以斜体显示在画布连线上（源端绿色/目标端紫色）
- IP 标签可独立拖拽

### 6.9 带宽与线缆长度

| 字段 | 说明 | 示例 |
|------|------|------|
| 带宽（Bandwidth） | 文本输入 | `1Gbps`、`10Gbps` |
| 线缆长度（Cable Length） | 文本输入 | `10m`、`50m` |
| 线缆描述（Description） | 多行文本 | 补充布线信息、编号等 |

## 7. 机柜系统

### 7.1 机柜尺寸预设

左侧边栏机柜区提供 4 种标准尺寸：

| 类型 | U 数范围 | 典型尺寸 | 适用场景 |
|------|----------|----------|----------|
| 壁挂式（Wall-mount） | ≤12U | 4U, 6U, 9U | 小型弱电间 |
| 小型（Small） | 13-22U | 18U | 分支机房 |
| 中型（Medium） | 23-36U | 27U | 楼宇机房 |
| 标准（Standard） | 42U | 42U | 数据中心主配线区 |

> 也支持自定义尺寸：6U, 9U, 12U, 18U, 22U, 27U, 32U, 36U, 42U, 45U, 47U

### 7.2 放置机柜

1. 在左侧边栏「网络机柜」区找到目标尺寸
2. 拖拽到画布上

### 7.3 在机柜中添加设备

**方式一：拖入**
- 从侧边栏拖拽设备，直接放到机柜节点上方
- 系统自动检测空闲 U 位并吸附

**方式二：右键菜单**
1. 右键点击机柜节点
2. 选择「添加设备到机柜」
3. 在弹出的设备选择器中选择目标设备

**机柜内移动：**
- 在机柜内上下拖拽设备，松开时自动吸附到最近 U 位
- 如果目标位置被占用，设备会弹回原位

### 7.4 U 位管理

- 每个设备占用整数 U 数（1U = 44.45mm 标准高度）
- 设备 U 高可在属性面板中调整（1-6U）
- 不同类别设备有默认 U 高（如服务器默认 2U，防火墙默认 1U）
- 系统自动检测空间冲突，防止设备重叠
- 柜满时会给出提示

### 7.5 机柜视图切换

机柜支持两种视图模式：

| 视图 | 显示内容 | 适用场景 |
|------|----------|----------|
| 正面视图（Front） | 设备面板 + 彩色类别标识 + U 位标签 | 查看设备布局和端口 |
| 背面视图（Back） | 电源模块 SVG 图示 + U 位标签 | 查看电源和布线情况 |

**切换方式：**
- 双击机柜节点
- 在右侧属性面板点击「正面」/「背面」按钮

### 7.6 机柜内设备属性

选中机柜内的设备节点，属性面板显示：

| 属性 | 说明 |
|------|------|
| U 高调整 | 滑块（1-6U），显示数字 |
| 电源数量 | 加减按钮（0-4），背面视图可见 |
| 节点颜色 | 颜色选择器 + 重置默认按钮 |
| 设备名称 | 文本输入 |
| 设备型号 | 文本输入 |
| 描述 | 多行文本 |

### 7.7 机柜配件

机柜配套提供以下配件：

| 配件 | 说明 |
|------|------|
| 理线器（Cable Manager） | 1U 高度的线缆管理槽 |
| 盲板（Blank Panel） | 遮挡空 U 位 |
| PDU（Power Distribution Unit） | 电源分配单元 |

### 7.8 删除机柜

- 选中机柜节点，按 Delete 键
- 机柜内的所有子设备会连同机柜一起被删除（级联删除）

## 8. 属性面板详解

属性面板位于窗口右侧（280px 固定宽度），根据当前选中的元素动态切换内容。

### 8.1 空状态

未选中任何元素时，显示提示文字「未选中任何元素」。

### 8.2 多选状态

选中 2 个或以上节点时：

- 显示选中设备数量（大号数字）
- 提示「按 Esc 取消全选」
- 提示「使用工具栏中的对齐工具」

### 8.3 标准设备节点属性

选中单个设备节点时，显示以下属性区域：

**节点颜色**
- 颜色选择器：更改节点背景色
- 「重置默认」按钮：恢复为分类默认颜色

**设备功能**
- 文本输入：自定义设备类别标签，覆盖默认分类名

**设备厂商**
- 文本输入：自定义厂商名称，覆盖数据库厂商

**设备名称**
- 文本输入：自定义节点显示名称，回车或失焦确认

**设备型号**
- 文本输入：自定义型号，覆盖数据库型号

**设备端口**
- 三组数字输入：RJ45 端口数 / SFP 光纤端口数 / 10G 光纤端口数（0-256）
- 下方实时显示端口组合预览
- 修改后节点 SVG 插图自动更新端口排布

**堆叠模式开关**
- 开关组件启用/禁用 STACK 端口
- 启用后节点底部显示金色 STACK 端口

**隧道端口**（仅 SDWAN CPE 设备）
- 开关启用/禁用隧道端口
- 启用后显示隧道端口数量调节（−/+ 按钮，1-8）
- 隧道端口自动上移避免与 STACK 端口重叠

**端口编号选项**
- 从零开始：端口从 GE0 开始编号（默认从 GE1 开始）
- 端口交错：多行端口按列交替编号

**设备照片**
- 上传区域：点击上传本地图片
- 预览区域：显示已上传照片
- 「删除照片」按钮
- 照片仅在属性面板和侧边栏缩略图中显示，不影响画布 SVG

**业务图片**（仅互联网应用设备）
- 多张图片上传和缩略图网格显示
- 每张图片可独立调节缩放（15%-300%滑块）
- 图片在节点内可自由拖拽定位
- 超出边界部分自动裁剪
- 悬停缩略图出现红色删除按钮

**IP 地址**
- 文本输入设备管理 IP

**描述**
- 多行文本输入自定义描述

**业务备注**
- 多行文本输入业务信息（如管理 IP / 上联端口 / VLAN 规划等）
- 鼠标悬停节点 3 秒后弹出此备注

**数据库描述**
- 该设备型号在数据库中的原始描述
- 可编辑，失焦后自动同步回数据库

### 8.4 机柜节点属性

| 属性 | 说明 |
|------|------|
| 机柜名称 | 文本输入，实时更新 |
| U 位统计 | 进度条 + 已用/空闲 U 数 |
| 空柜提示 | 引导文字：拖设备入柜 / 右键菜单添设备 |
| 视图切换 | 正面/背面按钮 |
| 机柜信息 | 总 U 数、总高度(px)、正面宽度、背面宽度 |

### 8.5 连线属性

参见[第 6 章连线系统](#6-连线系统)中各项配置的详细说明。

### 8.6 内联编辑

**无需打开属性面板**，直接在节点上修改：

1. 双击节点上的文本标签
2. 进入编辑模式，输入新文本
3. 按 Enter 确认，按 Esc 取消

**可内联编辑的字段（共 6 项）：**
- 设备功能
- 设备厂商
- 设备名称
- 设备型号
- 自定义名称
- 端口信息

**显示优先级**：自定义字段值 > 数据库字段值。如果自定义字段为空，则显示数据库默认值。

## 9. 文件管理

### 9.1 新建拓扑

1. 点击工具栏「新建」按钮（或 Ctrl+N）
2. 如果当前画布有未保存的更改，弹出确认对话框：
   - **保存**：先保存再新建
   - **不保存**：直接丢弃清空画布
   - **取消**：返回当前画布
3. 如果画布为空，直接清空开始新拓扑

### 9.2 保存拓扑

1. 点击工具栏「保存」按钮（或 Ctrl+S）
2. 首次保存弹出系统保存对话框，选择位置和文件名
3. 后续保存直接覆盖原文件
4. 文件格式为 `.topo.json`，结构如下：

```json
{
  "version": "1.1.0",
  "nodes": [...],
  "edges": [...],
  "viewport": { "x": 0, "y": 0, "zoom": 1 }
}
```

- 所有新增字段均为可选，保证向后兼容

### 9.3 打开拓扑

1. 点击工具栏「打开」按钮（或 Ctrl+O）
2. 如有未保存更改，弹出确认对话框
3. 选择 `.topo.json` 文件
4. 文件内容加载到画布，自动添加到最近文件列表

### 9.4 最近文件

- 文件菜单中动态显示最近打开的 10 个文件
- 点击直接打开
- 重复打开同一文件自动去重并置顶
- 存储在 `用户数据目录/recent-files.json`

### 9.5 自动保存

- 每 **2 分钟**自动保存一次（仅在画布有更改时触发）
- 如果当前文件已有路径，静默保存到该路径
- 如果没有路径（新建未保存），保存到自动保存备份位置
- **启动恢复**：如果上次退出时有自动保存文件，启动时弹出恢复对话框
  - 「恢复」：加载自动保存的内容
  - 「丢弃」：清除自动保存，空白画布

### 9.6 未保存更改指示器

工具栏 Topo 标题旁边有一个琥珀色圆点，表示当前画布有未保存的更改。保存后圆点消失。

## 14. 撤销与重做

| 操作 | 快捷键 | 工具栏按钮 |
|------|--------|------------|
| 撤销 | Ctrl+Z | ↶ 按钮 |
| 重做 | Ctrl+Y | ↷ 按钮 |

**工作原理：**
- 采用快照式历史栈，每次状态变更前自动保存完整画布快照
- 最大 **50 步**历史上限
- 可撤销到空白画布，也可重做到最新状态

**防抖优化：**
- **拖拽操作**：只在拖拽开始时保存快照，拖拽过程中不保存
- **滑块调节**：首次更改时保存快照，400ms 内连续更改不重复保存

## 20. 常见问题与技巧

### Q1: 如何快速复制一组设备？

1. 框选或 Shift+点击选中所有要复制的节点
2. 按 Ctrl+C 复制，再按 Ctrl+V 粘贴
3. 可以连续多次 Ctrl+V 生成多份，每次偏移 50px

### Q2: 如何对齐多个设备？

1. 选中需要对齐的设备（至少 2 个）
2. 工具栏左侧出现对齐工具按钮
3. 选择需要的对齐方式（左/居中/右/上/居中/下）
4. 对齐结果自动贴合 10px 网格

### Q3: 如何把设备放入机柜？

- **方法一**：从侧边栏直接拖设备到机柜节点上，自动放入空闲 U 位
- **方法二**：右键机柜 →「添加设备到机柜」→ 选择设备
- 机柜内设备可上下拖拽调整 U 位

### Q4: 自动保存如何工作？

- 每 2 分钟自动保存（仅在画布有更改时）
- 有路径则静默保存，无路径则存到备份位置
- 意外退出后，下次启动会提示恢复
- **建议**：新建拓扑后尽快手动保存一次，确保有路径

### Q5: 如何自定义设备外观？

- 修改节点颜色：选中设备 → 属性面板「节点颜色」
- 上传设备照片：属性面板「设备照片」区域（仅面板预览，不影响画布）
- 上传业务图片：互联网应用设备专属（支持多图/缩放/拖拽定位）
- 自定义名称/型号/厂商：直接在节点上双击文本编辑

### Q6: SDWAN 拓扑怎么画？

1. 从侧边栏拖入「SDWAN 节点」（云图标）到画布
2. 拖入「SDWAN CPE」设备到画布
3. 选中 CPE 节点 → 属性面板 → 开启「隧道端口」→ 设置隧道数量
4. 从 CPE 的隧道端口拖拽连线到 SDWAN 节点的左右连接点
5. 连线自动识别为 Tunnel 类型（青色阶梯线）
6. 可以再添加「互联网」、「公有云」、「数据中心」节点丰富拓扑

### Q7: 如何展示拓扑给客户/领导？

1. 切换到**鎏金主题**（工具栏 ✨ 按钮），更适合投影展示
2. 开启**Demo 演示模式**，自动展示拓扑
3. 导出 **PNG** 放到方案文档，或导出 **GIF** 展示动态效果
4. 导出 **PDF** 用于打印

### Q8: 不小心删错了怎么办？

按 Ctrl+Z 撤销即可恢复。撤销历史最多 50 步，足够覆盖大部分误操作。

### Q9: 如何共享拓扑文件？

- 将 `.topo.json` 文件发送给对方
- 对方用 Topo 打开即可查看完整拓扑
- 也可以导出为 PNG/PDF/GIF 作为不可编辑的图片分享
- 还可保存为模板，让对方通过模板导入

### Q10: 设备库中没有我需要的型号怎么办？

使用「添加自定义设备」功能（侧边栏底部按钮），填写分类、厂商、型号、端口信息后即可创建。创建的设备会保存到数据库中，下次打开仍然可用。
