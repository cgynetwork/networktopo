# Topo V1.5.0 发布说明

## 项目简介

Topo 是一款面向网络工程师的 Windows 桌面端网络拓扑绘制工具，基于 Electron + React + TypeScript 构建。支持从设备库拖拽创建拓扑、丰富的连线自定义、多格式导出，以及完整的文件管理。

## 系统要求

- Windows 10 / 11（x64）
- 无需安装额外依赖

## 安装方式

- **便携版**：下载 `Topo 1.0.0.exe`，双击运行即可
- **安装版**：下载 `Topo Setup 1.0.0.exe`，按向导安装，支持自定义安装目录

## 功能列表

### 设备库
- 7 大类设备：防火墙、交换机、无线控制器、无线接入点、服务器、PC 终端、笔记本终端
- 9 个厂商 42+ 预置型号（H3C、Huawei、Cisco、Ruijie、Aruba、Lenovo、Dell、HP、Apple）
- 自定义设备添加（分类、厂商、型号、端口、图片）
- 侧边栏实时搜索 + 右键编辑/删除

### 画布与节点
- 无限画布，滚轮缩放（10%~300%），中键平移
- 拖拽创建节点 + 增强 SVG 拟真渲染（渐变机壳、LED 指示灯、独立端口）
- 端口颜色编码（RJ45 暖灰 / SFP 冷蓝 / QSFP 深青 / MGMT 银灰）
- 设备图片上传（仅在属性面板预览）
- 双击编辑所有文字字段
- 框选 + Shift 多选 + 对齐/分布（6+2 种）
- 复制粘贴（Ctrl+C/V/D）
- 节点分组（Ctrl+G / Ctrl+Shift+G）
- 节点纵向缩放 + 端口面板自适应高度

### 连接线系统
- 4 种连接类型：网线、光纤、堆叠、无线
- 3 种动画：粒子流动、光带流动、信号波纹
- 3 种路径样式：自适应、直线、肘形
- 5 项外观自定义：粗细、颜色、速度、粒子大小、特效颜色
- 端口标签可拖拽 + 带宽/长度标注
- 堆叠模式（STACK 专线互连）

### 端口管理
- 动态端口渲染（基于 ports_info 字符串解析）
- 模块化端口编辑（RJ45/SFP/SFP28 数字输入）
- 端口编号选项（零基/交错）
- 智能分行布局（按宽度填行 + 左右分列）

### 文件管理
- .topo.json 保存/打开（向后兼容）
- 最近文件列表（最多 10 条）
- 2 分钟自动保存 + 启动恢复
- 拓扑模板（保存/加载/删除/导入）

### 导出
- PNG 图片导出
- PDF 导出（A4/A3）
- GIF 动画导出（保留 SVG 动画状态）

### 用户体验
- 撤销/重做（最大 50 步，Ctrl+Z/Y）
- 双主题（简洁白底蓝调 / 鎏金暗黑金调）
- 画布搜索（Ctrl+F）
- 网格/吸附切换
- 资产统计面板
- 业务备注 tooltip
- 确认对话框 + Toast 通知
- 底部状态栏
