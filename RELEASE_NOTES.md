# Topo V1.6.0 发布说明

## 项目简介

Topo 是一款面向网络工程师的 Windows 桌面端网络拓扑绘制工具，基于 Electron + React + TypeScript 构建。支持从设备库拖拽创建拓扑、丰富的连线自定义、多格式导出，以及完整的文件管理。

## 系统要求

- Windows 10 / 11（x64）
- 无需安装额外依赖

## 安装方式

- **便携版**：下载 `Topo 1.6.0.exe`，双击运行即可
- **安装版**：下载 `Topo Setup 1.6.0.exe`，按向导安装，支持自定义安装目录

## 功能列表

### 设备库（16 大类）
- SDWAN 节点、互联网、公有云、数据中心、SDWAN CPE
- 防火墙、交换机、无线控制器、无线接入点
- 服务器、PC 终端、笔记本终端
- 配线架、超融合、存储、运营商光猫
- 9 个厂商 42+ 预置型号（H3C、Huawei、Cisco、Ruijie、Aruba、Lenovo、Dell、HP、Apple）
- 自定义设备添加（分类、厂商、型号、端口、图片）
- 侧边栏实时搜索 + 右键编辑/删除

### 画布与节点
- 无限画布，滚轮缩放（10%~300%），中键/右键平移
- 拖拽创建节点 + 增强 SVG 拟真渲染（渐变机壳、LED 指示灯、独立端口）
- 端口颜色编码（RJ45 暖灰 / SFP 冷蓝 / 10GE+ 青色 / QSFP 深青 / MGMT 银灰 / WLAN 薄荷绿 / STACK 金色）
- 设备图片上传 + 互联网应用多图支持（缩放/拖拽/CliPath 裁剪）
- 双击编辑所有文字字段（设备功能/厂商/名称/型号/自定义名/端口信息）
- 框选 + Shift 多选 + 对齐（6 种）/分布（2 种）
- 复制粘贴（Ctrl+C/V/D）+ 分组（Ctrl+G / Ctrl+Shift+G）

### 连接线系统
- 5 种连接类型：以太网、光纤、堆叠、隧道、无线
- 3 种动画：粒子流动、光带流动、信号波纹
- 3 种路径样式：自适应贝塞尔、直线、阶梯（可调偏移）
- 外观自定义：粗细（1-10px）、颜色、动画速度（0.5x-4x）、粒子大小、特效颜色
- 连线方向切换 + 端口自动检测与手动选择
- 接口 IP 配置 + 带宽/线缆长度/描述标注
- 连接验证：Stack↔Stack、WLAN↔WLAN/AP、Tunnel↔SDWAN Node

### 机柜系统
- 壁挂/小型/中型/标准四类预设，支持 6U-47U 尺寸
- 设备拖入自动 U 位吸附 + 碰撞检测 + 满柜提示
- 正面面板视图 / 背面电源视图（双击切换）
- U 高调整（1-6U）+ 电源数量配置（0-4）
- 理线器/盲板/PDU 配件 + 级联删除

### 端口管理
- 动态端口渲染（基于 ports_info 字符串解析）
- 模块化端口编辑（RJ45/SFP/SFP28 数字输入，实时预览）
- 端口编号选项（零基/交错）
- 智能分行布局（按宽度填行 + 网络/光纤左右分列）

### 文件管理
- `.topo.json` 格式保存/打开（向后兼容）
- 最近文件列表（最多 10 条，去重置顶）
- 2 分钟自动保存 + 启动恢复提示
- 拓扑模板（保存/加载/删除/导入外部文件）

### 导出
- PNG 自动裁剪（24px 边距）
- PDF（A4/A3，自动横纵向）
- GIF 动画导出（40 帧，保留 SVG 动画）

### 用户体验
- 撤销/重做（最大 50 步，Ctrl+Z/Y，拖拽/滑块防抖）
- 双主题：默认白底蓝调 / 鎏金深色暖金调（即时切换，CSS 变量驱动）
- 画布全局搜索（Ctrl+F）+ 侧边栏设备搜索
- 网格显示 / 10px 吸附 / Demo 演示模式
- 资产统计面板（分类/型号/连线类型统计）
- 业务备注 tooltip（悬停 3 秒弹出）
- 确认对话框 + Toast 通知 + 底部状态栏
- 中英文界面切换
