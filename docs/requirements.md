# 网络拓扑绘制软件 - 需求规格说明

## 项目背景
面向网络工程师的 Windows 桌面端网络拓扑绘制工具，替代手工 WPS 绘制的低效方式。

## 核心需求

### R1 - 设备库管理
- R1.1 设备按分类展示：防火墙、交换机、无线控制器、无线接入点、服务器
- R1.2 分类支持折叠/展开
- R1.3 每个设备关联厂商和型号字段（如 H3C S5130S-28S-HPWR-EI）
- R1.4 设备以缩略图形式在左侧列表显示
- R1.5 默认使用 SVG 简约图标，支持用户替换为自定义图片
- R1.6 鼠标悬停显示设备详情弹窗（型号、端口数、描述等）
- R1.7 设备描述支持自由编辑（右键菜单）
- R1.8 搜索栏实时过滤设备
- R1.9 预置 H3C、Huawei、Cisco 等主流厂商常用型号

### R2 - 画布操作
- R2.1 无限画布，支持鼠标滚轮缩放（10%~300%）
- R2.2 支持拖拽平移画布
- R2.3 从侧边栏拖拽设备到画布创建节点
- R2.4 节点选中、移动、删除（Delete 键）
- R2.5 右键菜单操作节点
- R2.6 节点标签显示"名称 + 型号"，名称可编辑，型号只读

### R3 - 连接线系统
- R3.1 从设备连接点拖拽连线到另一设备
- R3.2 支持连接类型：网线（实线）、光纤（虚线）
- R3.3 支持显示状态切换：静态 / 动态
- R3.4 动态动画：粒子流动 + 光带流动，两种可切换
- R3.5 连线显示方向箭头，支持反转方向
- R3.6 选中连线后可编辑属性

### R4 - 数据存储
- R4.1 设备型号库存储在本地 SQLite 数据库
- R4.2 拓扑图保存为 .topo.json 文件
- R4.3 支持打开已有拓扑文件

### R5 - 导出
- R5.1 导出当前画布为 PNG 图片
- R5.2 导出为 PDF 文件（支持 A4/A3）

### R6 - UI 设计
- R6.1 主色调：黑白灰简约风格
- R6.2 强调色：淡蓝选中态 + 科技蓝动态效果
- R6.3 三栏布局：左侧设备库 + 中间画布 + 右侧属性面板

## 非功能需求
- NF1 运行在 Windows 10/11 平台
- NF2 操作响应流畅，动画帧率 ≥ 30fps
- NF3 打包为独立 .exe 安装包，无需用户安装额外依赖
