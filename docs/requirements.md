# 网络拓扑绘制软件 - 需求规格说明

## 项目背景
面向网络工程师的 Windows 桌面端网络拓扑绘制工具，替代手工 WPS 绘制的低效方式。

## 核心需求

### R1 - 设备库管理
- R1.1 设备按分类展示：防火墙、交换机、无线控制器、无线接入点、服务器、终端-PC、终端-笔记本
- R1.2 分类支持折叠/展开
- R1.3 每个设备关联厂商和型号字段（如 H3C S5130S-28S-HPWR-EI）
- R1.4 设备以缩略图形式在左侧列表显示
- R1.5 默认使用 SVG 简约图标，支持用户替换为自定义图片
- R1.6 鼠标悬停显示设备详情弹窗（型号、端口数、描述等）
- R1.7 设备描述支持自由编辑（右键菜单）
- R1.8 搜索栏实时过滤设备
- R1.9 预置 H3C、Huawei、Cisco、Ruijie、Aruba、Lenovo、Dell、HP、Apple 等主流厂商常用型号
- R1.10 侧边栏右键菜单：修改设备信息 / 删除设备
- R1.11 自定义设备添加（弹窗表单：分类、厂商、型号、描述、端口、图片）

### R2 - 画布操作
- R2.1 无限画布，支持鼠标滚轮缩放（10%~300%）
- R2.2 支持拖拽平移画布
- R2.3 从侧边栏拖拽设备到画布创建节点
- R2.4 节点选中、移动、删除（Delete 键）
- R2.5 右键菜单操作节点（复制、取消分组、删除）
- R2.6 节点标签显示"名称 + 型号"，所有字段可双击编辑
- R2.7 框选多设备（`selectionOnDrag`），支持 Shift+Click 多选
- R2.8 多选对齐工具（左/水平居中/右/顶/垂直居中/底，6 种）
- R2.9 多选分布工具（水平等距/垂直等距，2 种）
- R2.10 Escape 取消所有选中
- R2.11 网格显示/隐藏切换 + 10px 吸附开关

### R3 - 连接线系统
- R3.1 从设备连接点拖拽连线到另一设备
- R3.2 支持连接类型：网线（实线）、光纤（虚线）、堆叠（金色粗线）、无线（信号波纹）
- R3.3 支持显示状态切换：静态 / 动态
- R3.4 动态动画：粒子流动 + 光带流动 + 信号波纹，三种可切换
- R3.5 连线显示方向箭头，支持反转方向
- R3.6 选中连线后可编辑属性

### R4 - 数据存储
- R4.1 设备型号库存储在本地 SQLite 数据库
- R4.2 拓扑图保存为 .topo.json 文件
- R4.3 支持打开已有拓扑文件

### R5 - 导出
- R5.1 导出当前画布为 PNG 图片
- R5.2 导出为 PDF 文件（支持 A4/A3）
- R5.3 导出动画 GIF（捕获实际渲染帧，保留 SVG 动画状态）

### R6 - UI 设计
- R6.1 主色调：黑白灰简约风格（default 主题）
- R6.2 鎏金暗黑主题（gilded 主题），CSS 变量驱动一键切换
- R6.3 三栏布局：左侧设备库 + 中间画布 + 右侧属性面板
- R6.4 主题偏好 localStorage 持久化

---

### R7 — 连线自定义
- R7.1 线缆外观 5 项可调：粗细（1-10px）、颜色、动画速度（0.5-4×）、粒子大小（2-12px）、特效颜色
- R7.2 连接形式 3 种可选：自适应（bezier）、直线（straight）、肘形（step）
- R7.3 肘形连接线偏移量可调（10-400px）
- R7.4 连线数据保存到 .topo.json，打开后完整恢复

### R8 — 内联编辑与节点自定义
- R8.1 设备节点所有 6 个文字字段支持双击编辑（InlineEdit 组件）
- R8.2 字段：设备功能、设备厂商、产品型号、设备名称、自定义名称、端口信息
- R8.3 显示回退逻辑：`customField || device.field`，自定义值优先
- R8.4 节点底纹颜色自定义（customColor），属性面板颜色选择器 + 重置按钮

### R9 — 动态端口渲染
- R9.1 SVG 插图根据设备 `ports_info` 动态渲染端口（不再硬编码）
- R9.2 端口字符串解析引擎：`portParser.ts`，支持正则分割 + 语义分类
- R9.3 端口分类：网络端口（RJ45 电口）/ 千兆光纤（SFP）/ 万兆光纤（10GE+）/ QSFP / 管理口（MGMT）
- R9.4 连线时自动标注端口标签（源端蓝标 + 目标端橙标）
- R9.5 端口下拉选择器：列出设备所有可用端口供手动选择
- R9.6 端口左右分列布局：网络端口在左，光纤端口在右，按数量比例分配宽度

### R10 — 撤销/重做
- R10.1 快照式历史栈（最大 50 步，JSON 深拷贝）
- R10.2 所有状态变更操作前自动记录快照
- R10.3 拖拽防抖（仅开始时快照）、滑块防抖（首次记录，400ms 空闲重置）
- R10.4 支持撤销到空画布，重做到最新状态
- R10.5 Edit 菜单 Ctrl+Z / Ctrl+Y

### R11 — 最近文件与自动保存
- R11.1 最近文件列表自动维护（最多 10 条，`userData/recent-files.json`）
- R11.2 File 菜单动态显示最近文件，支持点击直接打开
- R11.3 重复文件自动去重并移到列表顶部
- R11.4 自动保存：2 分钟定时器，有路径时静默保存，无路径时写入 `autosave.topo.json`
- R11.5 启动时检测自动保存文件，弹出恢复确认对话框
- R11.6 工具栏 `•` 标记未保存状态

### R12 — 双主题系统
- R12.1 default（简洁白底蓝调）+ gilded（鎏金暗黑金调）
- R12.2 CSS 变量驱动：所有颜色通过 `--color-*` 变量定义，`data-theme` 属性控制
- R12.3 ThemeContext：React Context + localStorage 持久化
- R12.4 工具栏主题切换按钮，即时生效无刷新
- R12.5 MiniMap、滚动条、选中态等细节同步跟随主题

### R13 — 拟真设备渲染
- R13.1 增强 SVG 插图（144px 高度），渐变机壳 + LED 指示灯 + 前面板端口
- R13.2 5 种设备类型各异：交换机（LED 阵列 + 端口区）、防火墙（安全面板 + 盾牌图标）、AC（信号波浪弧线）、AP（天线 + 发光）、服务器（硬盘托架 4×3 + PSU + 风扇）
- R13.3 端口颜色编码：RJ45（暖灰）、SFP（冷蓝）、QSFP（深青）、MGMT（银灰）、WLAN（青绿）、STACK（金色）
- R13.4 设备图片上传功能（存储于 `userData/device-images/`）
- R13.5 图片在属性面板预览，画布始终使用 SVG 渲染

### R14 — 端口布局算法
- R14.1 按宽度填行：逐个端口检测可用宽度，放不下自动换行
- R14.2 每行端口水平居中，端口面板垂直居中
- R14.3 端口间 3px 间距，端口组间 10px 间距
- R14.4 SVG 高度响应式：纵向缩放节点时端口面板自动扩展
- R14.5 模块化端口编辑：三个数字输入（网络端口 / 千兆光纤 / 万兆光纤数量）
- R14.6 端口顺序编号（GE1/GE2/...）
- R14.7 端口编号选项：零基（GE0 起始）+ 交错（列优先交替行）

### R15 — 设备堆叠
- R15.1 设备底部 STACK 端口开关
- R15.2 STACK 端口专线互连（拒绝 STACK ↔ 非 STACK）
- R15.3 金色粗线缆（7px）+ 专用慢速动画
- R15.4 堆叠状态保存到 .topo.json

### R16 — 无线与终端设备
- R16.1 PC 终端和笔记本终端两个新分类
- R16.2 终端设备含 WLAN 无线端口 + GE 网络端口
- R16.3 无线连接类型（wave 信号波纹动画）
- R16.4 WLAN ↔ WLAN 允许，WLAN ↔ 无线 AP 允许，WLAN ↔ 其他拒绝

### R17 — 业务备注
- R17.1 设备属性面板「业务说明」多行文本编辑
- R17.2 鼠标悬浮设备节点 3 秒后显示业务说明 tooltip
- R17.3 tooltip 支持管理 IP、上联接口、业务规划、VLAN 等信息

### R18 — 复制粘贴
- R18.1 Ctrl+C 复制选中节点到画布剪贴板
- R18.2 Ctrl+V 粘贴（偏移 50px），支持重复粘贴
- R18.3 Ctrl+D 快速原地复制（偏移 30px）
- R18.4 画布右键菜单含粘贴选项

### R19 — 拓扑模板
- R19.1 保存画布为拓扑模板（存储于 `userData/templates/`）
- R19.2 模板列表加载、删除
- R19.3 从外部 .topo.json 文件导入模板
- R19.4 模板名称通过 PromptDialog 弹窗输入（替代不可靠的 `window.prompt()`）

### R20 — 画布搜索
- R20.1 Ctrl+F 呼出浮动搜索栏
- R20.2 按设备名称/型号/厂商/分类/描述/IP/备注实时检索
- R20.3 点击搜索结果自动定位并选中设备

### R21 — 节点分组
- R21.1 Ctrl+G 为选中设备分配分组名称
- R21.2 节点底部显示彩色分组标签（📁 badge），颜色按名称哈希分配
- R21.3 Ctrl+Shift+G 或右键菜单取消分组
- R21.4 分组信息保存到 .topo.json

### R22 — 画布辅助
- R22.1 工具栏缩放控件（+/- 按钮 + 百分比显示 + 适应视窗按钮）
- R22.2 底部状态栏：实时缩放比例、设备/连线计数、选中数量、快捷键提示
- R22.3 画布右键菜单：粘贴/全选/适应视窗
- R22.4 资产统计面板：工具栏悬浮提示按分类和型号汇总设备/连线

---

## 非功能需求

- NF1 运行在 Windows 10/11 平台（x64）
- NF2 操作响应流畅，动画帧率 ≥ 30fps
- NF3 打包为独立 .exe 安装包和便携版，无需用户安装额外依赖
- NF4 .topo.json 格式向后兼容（新字段均为可选，旧文件正常加载）
- NF5 双入口架构：`main.js`（生产 CommonJS 入口）+ `src/main/*.ts`（electron-vite dev 入口），功能同步
