# 设计规范

## UI 设计原则
- **简洁直观**：减少层级，常用操作不超过 2 步
- **黑白灰基调**：大面积留白，减少视觉疲劳
- **功能优先**：装饰性元素最小化

## 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#FFFFFF` | 画布区域 |
| 次背景 | `#FAFAFA` | 侧边栏、属性面板 |
| 边框 | `#E5E5E5` | 分割线、面板边框 |
| 主文字 | `#1A1A1A` | 标题、正文 |
| 次文字 | `#6B7280` | 描述、辅助信息 |
| 选中态 | `#E3F2FD` | 选中节点背景 |
| 选中边框 | `#2196F3` | 选中节点边框 |
| 流量动画 | `#2196F3` | 动态连线光带 |
| 悬停态 | `#F3F4F6` | 列表项悬停 |
| 删除/危险 | `#EF4444` | 删除按钮 |

## 布局规范

- 侧边栏宽度：260px（固定）
- 属性面板宽度：280px（固定）
- 工具栏高度：48px
- 最小窗口尺寸：1024×768
- 侧边栏设备缩略图：64×64px

## 字体规范

| 用途 | 字号 | 字重 |
|------|------|------|
| 分类标题 | 14px | 600 (Semibold) |
| 设备名称 | 13px | 500 (Medium) |
| 设备型号 | 12px | 400 (Regular) |
| 节点名称 | 12px | 500 (Medium) |
| 节点型号 | 10px | 400 (Regular) |
| 工具栏按钮 | 13px | 500 (Medium) |

- 字体族：`"Microsoft YaHei", -apple-system, sans-serif`

## 图标规范

设备类型默认 SVG 图标尺寸：
- 侧边栏缩略图：48×48px（内含图标 32×32px）
- 画布节点：图标 40×40px

设备类型图标示意：
- 防火墙：盾牌造型
- 交换机：矩形 + 端口阵列
- 无线控制器：天线 + 信号波
- 无线接入点：圆形 + 信号波
- 服务器：机架式矩形

## 交互规范

- 拖拽：从侧边栏拖出设备，拖入画布放置
- 连线：从设备边缘连接点（Handle）拖出线到目标设备
- 缩放：Ctrl + 滚轮 或 工具栏按钮
- 平移：鼠标中键拖拽 或 空白区域拖拽
- 删除：选中后按 Delete 键
- 右键：上下文菜单
