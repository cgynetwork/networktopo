# 设计规范

## UI 设计原则
- **简洁直观**：减少层级，常用操作不超过 2 步
- **黑白灰基调**：大面积留白，减少视觉疲劳（default 主题）
- **鎏金暗黑**：深色暖金调，适合弱光环境（gilded 主题）
- **功能优先**：装饰性元素最小化

## 布局规范

- 侧边栏宽度：260px（固定）
- 属性面板宽度：280px（固定）
- 工具栏高度：48px
- 状态栏高度：24px
- 最小窗口尺寸：1024×768
- 默认窗口尺寸：1400×900
- 侧边栏设备缩略图：64×64px

## 图标规范

设备类型默认 SVG 图标尺寸：
- 侧边栏缩略图：48×48px（内含图标 32×32px）
- 画布节点 SVG 插图：144px 高度（响应式，最小 100px）
- 设备节点总高度 = header(~30px) + info(~60px) + SVG 插图 + padding(~12px)

设备类型图标示意：
- 防火墙：盾牌 + 安全面板 + 端口区
- 交换机：LED 阵列 + 前面板端口区
- 无线控制器：信号波浪弧线 + 端口区
- 无线接入点：天线 + 发光效果 + 安装支架
- 服务器：硬盘托架（4×3 矩阵）+ 右侧 NIC 面板
- PC 终端：显示器图标 + 端口区
- 笔记本终端：笔记本图标 + 端口区

## 组件设计

### 分组标签（Group Badge）
- 位置：设备节点底部，端口区域之下
- 样式：圆角 pill（`rounded`），`px-2 py-0.5`，📁 图标 + 分组名
- 颜色：背景 `hsl(hash(name) % 360, 45%, 88%)`，文字 `hsl(hash(name) % 360, 55%, 28%)`
- 同组设备颜色一致（相同 hash 值）

### 搜索浮层（Search Overlay）
- 位置：画布顶部居中，`z-40`
- 浮动条：输入框 + 🔍 图标 + 清空按钮
- 结果列表：`max-h-64 overflow-y-auto`，每项显示设备名 + 型号/厂商
- 点击结果自动定位画布节点

### 状态栏（Status Bar）
- 高度 24px，固定在画布和工具栏之间
- 左侧：缩放比例 + 设备计数 + 连线计数 + 选中计数
- 右侧：常用快捷键提示（Ctrl+G/F/C/V、Esc）

### PromptDialog（文本输入对话框）
- 替代 `window.prompt()` 的 React 模态弹窗
- 固定遮罩层 + 居中卡片 + 自动聚焦输入框
- Enter 确认 / Escape 取消 / 点击遮罩取消

### Toast 通知
- 位置：右下角固定，`z-50`
- 类型：success（绿）、error（红）、info（蓝）
- 自动 3 秒消失，带进度条动画

## 交互规范

- 拖拽：从侧边栏拖出设备，拖入画布放置
- 连线：从设备端口 Handle 拖出线到目标设备端口
- 缩放：Ctrl + 滚轮 或 工具栏 +/- 按钮（10%~300%）
- 平移：鼠标中键拖拽 或 空白区域左键拖拽
- 删除：选中后按 Delete/Backspace 键
- 右键：节点右键菜单 / 批量右键菜单 / 连线右键菜单 / 画布右键菜单 / 侧边栏右键菜单
- 多选：Shift+Click 追加选择 / 画布拖拽框选 / Ctrl+A 全选
- 编辑：双击节点文字字段进入编辑模式（Enter 确认 / Esc 取消）
- 分组：Ctrl+G 创建分组 / Ctrl+Shift+G 或右键菜单取消分组
- 搜索：Ctrl+F 呼出画布搜索浮层
- 撤销重做：Ctrl+Z / Ctrl+Y
- 复制粘贴：Ctrl+C/V 复制粘贴 / Ctrl+D 原地复制
